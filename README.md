# SimpleChessEngine

A chess engine written in C++

Main features:
* Minimax search with alpha beta pruning
* Zobrist hashing
* Bitboards for move generation
* Iterative deepening based on a timeout

Under development:
* Predictive branching
* Openings
* Tablebase
